小飞管理面板
#小飞设置
功能设置
  • 小飞点歌作为默认点歌 #小飞设置点歌 + 开启/关闭 {{@music}}
    如有多个点歌插件，开启后将使用小飞点歌作为的默认点歌(重启生效)
  • 小飞天气作为默认天气 #小飞设置天气 + 开启/关闭 {{@weather}}
    如有多个天气插件，开启后将使用小飞天气作为的默认天气(重启生效)
  • 戳一戳推荐歌曲 #小飞设置戳一戳 + 开启/关闭 {{@poke}}
    开启后戳一戳机器人将自动为您推荐一首歌
  • 默认搜索点歌 #小飞设置搜索点歌 + 开启/关闭 {{@is_list}}
    开启后使用点歌指令将默认返回搜索点歌列表
  • 默认音乐源 #小飞设置默认音乐源 + （QQ、网易、酷我、酷狗） {{@music_source}}
    设置【#点歌 #小飞点歌】指令的默认音乐源
  • 高品质点歌 #小飞设置高品质点歌 + 开启/关闭 {{@music_high_quality}}
    开启后（QQ、网易、酷我源）点歌将返回高品质音乐
  • 卡片天气 #小飞设置卡片天气 + 开启/关闭 {{@card_weather}}
    新版QQ天气Ark，PC暂不支持查看（开启后会与图片天气同时发送）
